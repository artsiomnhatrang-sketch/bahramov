# Инструкция: завершение сессии

После того как последняя статья написана и запушена, выполни следующее (обнови файл STATUS.md в корне, создай его если нет):

1. Дата и время последнего обновления.
2. Всего статей опубликовано (взять из CLAUDE.md).
3. Последние 3 статьи: файл + H1 + дата публикации.
4. Что в работе / не завершено (незакоммиченное, ожидает проверки диффа, ручная дистрибуция и т.п.).
5. Очередь тем из раздела TODO в CLAUDE.md.
6. Открытые хвосты и техдолг.
7. Правила проекта в 1-2 строках (сверка на дубль перед статьёй; фактчек; дифф перед пушем; publish-article.sh запускает пользователь вручную).

Формат STATUS.md — компактный, читаемый с первого взгляда.
После обновления STATUS.md: git add STATUS.md, коммит "docs: update session status", запушь.
